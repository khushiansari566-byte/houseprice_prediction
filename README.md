🏠 House Price Prediction

📌 Project Overview

This project predicts house prices using Machine Learning and Deep Learning techniques.
It uses the California Housing dataset and compares multiple models to find the best performance.

📊 Dataset

- California Housing Dataset
- Features:
  - Longitude & Latitude
  - Housing Median Age
  - Total Rooms & Bedrooms
  - Population & Households
  - Median Income
  - Ocean Proximity (categorical)

🧠 Models Used

- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor
- Artificial Neural Network (ANN)

📈 Model Performance

Model| R² Score
Linear Regression| 0.63
Random Forest| 0.73
Gradient Boosting| ~0.75
ANN (Deep Learning)| ~0.75

📈 Conclusion

For structured/tabular data, traditional machine learning models can perform as well as or better than deep learning models.
